# Sensor Data Assets

This directory contains assets related to sensor simulation for Module 2.

## Contents

- `lidar_pointcloud_example.png` - Example of LiDAR point cloud data
- `depth_camera_output.png` - Example of depth camera RGB-D output
- `imu_orientation_data.png` - Visualization of IMU orientation data
- `sensor_fusion_diagram.png` - Diagram showing sensor fusion architecture
- `lidar_camera_calibration.png` - Example of LiDAR-camera calibration target
- `multi_sensor_integration.png` - Diagram of multi-sensor integration

These assets support the documentation in Chapter 3: Sensor Simulation.
