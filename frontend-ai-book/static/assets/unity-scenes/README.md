# Unity Scene Assets

This directory contains assets related to Unity rendering scenes for Module 2.

- `unity_robot_visualization.png` - Screenshot of Unity robot visualization
- `gazebo_unity_synchronization.png` - Diagram showing Gazebo-Unity synchronization
- `hri_interface_example.png` - Example of human-robot interaction interface
- `unity_rendering_pipeline.png` - Diagram of Unity rendering pipeline for robotics
- `ros_tcp_connection.png` - Diagram showing ROS-TCP connection architecture

These assets support the documentation in Chapter 2: High-Fidelity Unity Rendering.
